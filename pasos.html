Pasos para trabajar con git

Primero tenemos que meternos en Git desde la Web y aceptar que nos estan compartiendo un proyecto. Despues de esto podremos descargarnos el proyecto desde nuestro ordenador abriendo la consola y escribiendo el comando "Git clone". Ya podemos crear o modificar en el proyecto. El paso siguiente seria crear una rama con checkout. Añadir "Git add ., Añadir el comando Git commit -m escribiendo lo que hemos hecho y lanzarlo con Git push. Nos vamos a la Web de Git para hacer un pull request y esperar la aceptacion de los compañeros para el merge. Despues podremos descargarnos el proyecto final escribiendo en la consola "git checkout master" para cambiar a master y "git pull" para volcarnos los nuevos.
